## uuid

生成uuid

### Usage

```ts
import { uuid } from '@etils/tool';
uuid();
```

### 文档

#### 方法

| 方法名 | 参数 | 返回值 | 说明     |
| ------ | ---- | ------ | -------- |
| uuid   | -    | string | 生成uuid |

### 例子

```ts
import { uuid } from '@etils/tool';
uuid(); // af35f9a7-5d5a-42a0-adb2-8b4ae337b228
```
