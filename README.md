# etils
javascript工具库


## 安装

```bash
pnpm add @etils/core
```

## 使用

``` ts
import { downloadByUrl } from "@etils/tool"
downloadByUrl('https://www.baidu.com/img/PCtm_d9c8750bed0b3c7d089fa7d55720d6cf.png', 'baidu.png')
```
